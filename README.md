# 简述 admin

## 这是是用Vue开发的博客后台管理系统，具体模块有：

1. 登录、退出
2. 文章管理
   - 文章修改、删除
   - 添加文章
   - 草稿箱
3. 笔记管理
   - 笔记添加
   - 笔记修改、删除
4. 标签管理
   - 标签添加
   - 标签修改、删除
5. 项目管理
   - 项目添加
   - 项目修改、删除
   - 草稿箱


>  [在线演示地址](http://admin.lisfes.cn)

> 本项目由 [Wlisfes](https://github.com/Wlisfes) 开发，您可以随意修改、使用！！

# 如何安装与使用

```
git clone https://github.com/Wlisfes/admin.git

cd admin

npm install     安装依赖

npm run dev     运行

npm run build   打包
```
